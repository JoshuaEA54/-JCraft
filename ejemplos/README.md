# Ejemplos de :JCraft

Esta carpeta contiene programas de ejemplo completos escritos en :JCraft para demostrar diferentes características del lenguaje.

## 📁 Archivos de Ejemplo

| Archivo | Descripción | Conceptos |
|---------|-------------|-----------|
| `fibonacci.jcraft` | Secuencia de Fibonacci recursiva | Recursividad, loops, funciones |
| `numeros_primos.jcraft` | Encontrar números primos hasta 50 | Funciones booleanas, while loops, lógica |
| `analisis_datos.jcraft` | Análisis estadístico de una lista | Funciones múltiples, listas, operaciones |
| `juego_palabras.jcraft` | Sistema de puntuación simple | Mapas/diccionarios, condicionales |
| `demo_compound_ops.jcraft` | Operadores compuestos (+=, -=, etc.) | Operadores de asignación compuesta |
| `ejemplo_inventario.jcraft` | Sistema de inventario | Mapas, operaciones sobre colecciones |
| `ejemplo_estructura.jcraft` | Estructura básica de programa | Variables globales, funciones, main |
| `guia_if_else.jcraft` | 10 ejemplos de if/else | Condicionales, comparadores, dispensadores |

## 🚀 Cómo Ejecutar

### Desde el IDE
1. Ejecuta `python main.py`
2. Abre un archivo desde "Archivo → Abrir"
3. Presiona "▶ Ejecutar" o F5

### Desde línea de comandos
```bash
python -c "from lang.interpreter import run_source; print(run_source(open('ejemplos/fibonacci.jcraft').read()))"
```

## 📚 Por Nivel de Dificultad

### 🟢 Principiante
- `ejemplo_estructura.jcraft` - Estructura básica
- `demo_compound_ops.jcraft` - Operadores simples
- `guia_if_else.jcraft` - Condicionales

### 🟡 Intermedio
- `ejemplo_inventario.jcraft` - Uso de mapas
- `juego_palabras.jcraft` - Lógica de juego simple
- `numeros_primos.jcraft` - Algoritmos básicos

### 🔴 Avanzado
- `fibonacci.jcraft` - Recursividad
- `analisis_datos.jcraft` - Múltiples funciones complejas

## 💡 Aprender Paso a Paso

1. **Empieza con** `ejemplo_estructura.jcraft` para entender la estructura básica
2. **Practica condicionales** con `guia_if_else.jcraft`
3. **Explora funciones** con `numeros_primos.jcraft`
4. **Aprende recursividad** con `fibonacci.jcraft`
5. **Domina colecciones** con `analisis_datos.jcraft`

## 🎯 Características Destacadas

Cada ejemplo demuestra al menos una característica clave:

- ✅ Declaración de variables con tipos
- ✅ Funciones con retorno y sin retorno
- ✅ Estructuras de control (if, while, for)
- ✅ Operaciones con colecciones (listas y mapas)
- ✅ Entrada/salida con `letrero()` y `cofre("Que ingresar aquí")`
